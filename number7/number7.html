Some bees get the nectar soon, others have to struggle

What inequity does this system model?

This system shows structural inequality where the same effort does not guarantee the same progress. Although both receive similar input, only one consistently converts effort into advancement, reflecting how systems can benefit the privileged.

How does time (or repetition) make the difference more visible?

Over repeated interactions, the small disadvantage compounds into a visible gap in outcomes. Time amplifies the hidden imbalance, showing how consistent access to progress creates momentum while uncertainty creates delay.

What did you learn / What are your reflections on this process?

I found that inequality can come from just one rule that quietly shifts how we reward effort. This project highlights how systems can seem fair at first glance,but they can actually create unequal outcomes when you look a little deeper.
